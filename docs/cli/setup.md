# `autocrab setup`

Initialize `~/.autocrab/autocrab.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
autocrab setup
autocrab setup --workspace ~/.autocrab/workspace
```

To run the wizard via setup:

```bash
autocrab setup --wizard
```
